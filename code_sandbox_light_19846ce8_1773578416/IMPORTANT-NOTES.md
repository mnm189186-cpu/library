## 🟡 تحذيرات مهمة

### ⚠️ عدم نشر مفاتيح API علناً

**لا تضع أبداً:**
- ❌ Gemini API Key في كود JavaScript
- ❌ Blog ID السري
- ❌ OAuth Tokens

**استخدم بدلاً من ذلك:**
- ✅ Apps Script كـ Proxy
- ✅ Script Properties في Apps Script
- ✅ Environment Variables

---

### ⚠️ حدود الاستخدام المجاني

**Google Drive API:**
- 1 مليار استعلام/يوم (كافٍ جداً)
- حد تخزين: 15 GB

**Blogger API:**
- 10,000 طلب/يوم (كافٍ)

**Gemini API (مجاني):**
- 60 طلب/دقيقة
- 1500 طلب/يوم
- 1 مليون token/شهر

**إذا تجاوزت الحدود:** سيتوقف الخدمة مؤقتاً

---

### ⚠️ OCR ودقة النتائج

**عوامل تؤثر على جودة OCR:**
- ✅ **جودة PDF عالية** = نتائج أفضل
- ❌ **PDF منخفض الجودة** = نتائج ضعيفة
- ✅ **نص واضح** = استخراج دقيق
- ❌ **خط يدوي** = نتائج سيئة
- ✅ **PDF نصي** (Text-based) = ممتاز
- ❌ **PDF صور فقط** = متوسط

**نصيحة:** استخدم PDF بدقة 300 DPI على الأقل

---

### ⚠️ مشاكل CORS المحتملة

**إذا واجهت خطأ CORS:**

```
Access to fetch at '...' from origin '...' has been blocked by CORS policy
```

**الحلول:**
1. استخدم Apps Script كـ Proxy
2. تأكد من نشر Web App بإعداد "Anyone"
3. استخدم Google Drive مع مشاركة عامة

## 🚨 مشاكل معروفة وحلولها

### المشكلة 1: القالب لا يحفظ

**السبب:** XML غير صالح
**الحل:**
- تحقق من إغلاق جميع الوسوم
- تحقق من عدم وجود أحرف خاصة غير معروفة
- استخدم Blogger Theme Validator

### المشكلة 2: مكتبة الكتب فارغة

**السبب:** رابط JSON خاطئ أو CORS
**الحل:**
- افتح Console (`F12` > Console)
- تحقق من الأخطاء
- تأكد من رابط JSON صحيح وعام

### المشكلة 3: رفع PDF لا يعمل

**السبب:** Apps Script غير منشور أو صلاحيات خاطئة
**الحل:**
- تأكد من نشر Apps Script كـ Web App
- تحقق من "Who has access" = Anyone
- جرّب Re-deploy

### المشكلة 4: OCR لا يستخرج النص

**السبب:** PDF غير نصي أو بجودة منخفضة
**الحل:**
- استخدم PDF بجودة أعلى
- تأكد من أن PDF يحتوي على نص (وليس صوراً فقط)
- جرّب لغة OCR مختلفة

### المشكلة 5: دردشة AI لا ترد

**السبب:** API Key خاطئ أو حد الاستخدام
**الحل:**
- تحقق من صحة API Key
- تحقق من Script Properties في Apps Script
- تحقق من حدود الاستخدام اليومية
